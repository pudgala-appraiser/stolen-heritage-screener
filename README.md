# 도난국가유산 스크리너

국가유산청이 공개한 도난 국가유산 정보(2025년 7월말 기준 정본 511건 + 누리집 갱신분)를
내장한 1차 대조 도구입니다. 기관·지자체의 취득(구입·기증) 심사 목록이나 반출 감정 물품을
도난 기록과 대조해 검토 후보를 제시합니다.

## 사용

- 브라우저에서 바로 사용: **(Pages 주소 — https://pudgala-appraiser.github.io/stolen-heritage-screener)**
- 또는 `index.html`을 내려받아 더블클릭으로 실행합니다. 대조 기능은 인터넷 없이도 동작합니다.

## 이 도구의 성격 (중요)

- 판정자가 아니라 **1차 스크리너**입니다. 특정 유물을 "도난품"이라 단정하지 않으며,
  후보와 근거·유사도를 제시하고 최종 판단은 감정위원이 합니다.
- **'통과'는 공개 도난 기록과의 접점 부재**를 뜻할 뿐, 도난품이 아님의 증명이 아닙니다.
  미신고 도난품, DB 기준일 이후 신규 공고, 이미지로만 식별 가능한 경우는 범위 밖입니다.
- 화면에 올리는 질의 자료(목록·사진)는 **브라우저 밖으로 전송되지 않습니다**.

## 데이터

- 출처: 국가유산청 발행 『도난국가유산 정보』(2025) 및 누리집 '도난 국가유산 정보'
  게시판(공개 행정정보).
- 회수 상태를 반영합니다 — 잔여 수량이 0으로 확인된 건은 경보 대상에서 제외됩니다.

## 갱신

1. 누리집에 새 공고가 뜨면: 도구의 **DB 관리·발행** 탭에서 등록(도난/유실 구분)
2. **갱신 스크리너 발행**으로 새 HTML을 내려받기
3. 그 파일의 이름을 `index.html`로 바꿔 이 저장소에 다시 업로드(교체 커밋)
   — 접속 주소는 그대로, 내용만 갱신됩니다.
